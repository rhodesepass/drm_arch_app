# EEZ Studio Design Project for drm_arch_app

EEZ version: `V0.25.1`

This directory stores the EEZ Studio project, helper scripts for font and icon export, and editor state files.

## Export Rules

- EEZ project file: `epass.eez-project`
- Current export target: `../../drm_arch_app/generated_ui/`
- Runtime code only builds `drm_arch_app/generated_ui/*`

Do not edit `generated_ui/*` directly. Update the EEZ project and export again instead.
